# CodeOrganizer
Code Organizer an android app that helps you to organize your codes 


## Features 
* offline usage
* local database (use's your phones storage)
* Categorization
* User Friendly 

## Stack
* Android Studio with kotlin
* Sqlite database (free database , stores content locally )
